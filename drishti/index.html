DRISHTI//NARAD
read-only unreadable file scanner
What this does: scans a folder you choose and reports which files fail to open or read — nothing is deleted, moved, or modified. Review the list yourself and decide what to do with each file.
Not supported in this browser. This tool needs the File System Access API, currently available in Chrome, Edge, and other Chromium-based browsers on desktop. Firefox and Safari can't grant folder access to a web page yet.
Scan mode
Quick check
Tries opening each file and reading a small sample. Fast — good first pass.
Thorough scan
Reads every file fully, start to end. Slower, catches errors mid-file.
