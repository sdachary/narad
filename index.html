NARAD//TOOLS
standalone browser utilities
SANDESH
Local network file transfer via WebRTC. Pair two devices with a 6-digit code and send files directly browser-to-browser.
DRISHTI
Read-only file scanner. Point it at a folder and it reports which files fail to open — nothing is modified.
SHUDDHI
Clean a compromised Windows PC. Step-by-step guide with a built-in scanner script — no tech knowledge needed.
BADHAI
Extract audio from video files in your browser. Drop a video, pick MP3/WAV/AAC/OGG, download. Nothing leaves your machine.
All tools run entirely in your browser. Nothing is uploaded or stored remotely.
