Karma — Key Features

Core capabilities powering the compute marketplace

🏪
Compute Marketplace
Task queue, contributor portal, and routing engine that connects buyers of compute with verified node providers. Supports CPU, GPU, and general-purpose workloads.
💳
UPI + Escrow Payments
RBI-compliant escrow holds funds until task completion. UPI and bank transfer for payouts. Minimum ₹100 payout threshold with daily settlement.
⭐
Trust Score System
3-tier node scoring: Verified (95–100), Trusted (80–94), New (0–79). Priority routing to high-trust nodes. Scores built from task success rate, uptime, and quality metrics.
🔀
Hybrid Routing Engine
Intelligent task routing: Verified internal nodes first → Trusted internal → External providers (Golem/Akash) as fallback. Optimises cost and latency automatically.
📊
GST + TDS Compliance
Automated 18% GST invoicing (SAC 998313), TDS deduction at 10% for payouts > ₹30K, Form 16A generation, and Chitragupta integration for tax reporting.
🛡️
Subagent Verification
Multi-layer verification framework with Compliance Checker, Security Auditor, Quality Reviewer, and Payment Validator agents running automated checks on every task and payout.

Risk Management

Key risks, mitigations, and compliance requirements across both platforms

Regulatory Risk
Impact: High · Probability: Medium
DPDP Act 2023, RBI/SEBI mandates evolving. New CERT-In VPN rules create compliance surface.
Mitigation: Legal consultation, dedicated compliance team, proactive posture on data localisation.
Data Sovereignty (Vishwakarma)
Impact: High · Probability: High (current state)
Supabase US-hosted Postgres and Vercel place Indian user data outside jurisdiction.
Mitigation: Phase 2 migration to self-hosted India Supabase; dual-run validation before cutover.
Data Breach
Impact: High · Probability: Low
Single-point Supabase compromise or OCI VM breach could expose user data.
Mitigation: End-to-end encryption, HashiCorp Vault for secrets, access control, ISO 27001 target.
Node Dropout (Karma)
Impact: Medium · Probability: Medium
Self-hosted contributors may go offline causing task failures.
Mitigation: Multi-provider fallback (Golem/Akash), retry logic, trust-score-based routing, redundant nodes.
Payment Failure
Impact: High · Probability: Low
UPI gateway outages or escrow failures could delay contributor payouts.
Mitigation: Escrow guarantee, backup payment providers, 100% payment loss coverage per insurance table.
AI Quality Regression
Impact: Medium · Probability: Medium
Switching from Groq/Anthropic to Ollama may reduce inference quality for complex tasks.
Mitigation: A/B testing, Anthropic API retained as premium fallback, gradual traffic migration.
Market + Contributor Churn
Impact: Medium · Probability: Medium
Karma contributors exit or VPN/storage subscribers churn to competitors.
Mitigation: Multi-provider network, trust-score rewards, India-first pricing, no-lock-in positioning.
Hyderabad Power / Connectivity
Impact: Medium · Probability: Low (managed)
Home server option vulnerable to power fluctuations and single ISP.
Mitigation: APC 1500VA UPS, Airtel/Jio Business symmetric fibre + 4G/5G LTE failover router.

Indian Government Compliance

Regulatory requirements applicable to both Karma and the Sovereign Stack

GST Act, 2017
18% on IT services. SAC code 998313 (IT consulting/software), 998315 (cloud hosting). Register above ₹20L turnover; OIDAR registration regardless of turnover for overseas services.
CBIC · 18%
TDS — Section 194C/J
Deduct TDS at 10% on Karma contributor payouts above ₹30K per transaction per FY. Issue Form 16A certificates quarterly. File 27A returns.
Income Tax Dept
DPDP Act, 2023
Digital Personal Data Protection — requires explicit consent for data collection, mandates data localisation for sensitive personal data. Drives the entire India infrastructure migration.
MeitY
RBI Payment Guidelines
2FA required on all digital payment transactions. KYC/Aadhaar verification for UPI. Escrow account for marketplace task payments until completion.
RBI Directions
CERT-In VPN Mandate
Section 70B requires VPN providers to log user data. Public legal challenge posture planned. Publishing explicit no-logs policy with India jurisdiction-only stance is the core differentiator.
CERT-In 2022
KYC Norms
Aadhaar + phone verification for contributor onboarding. Required for UPI payout eligibility and TDS certificate issuance. Integrated into Karma contributor portal registration flow.
