How to connect Google Sheets 1. Open your sheet → File → Share → Publish to web
2. Choose the sheet tab, select format: Comma-separated values (.csv)
3. Click Publish and copy the link
4. Paste the link below and click Fetch
The sheet must be published publicly. Your data never leaves your browser.
How to connect Zoho Sheet 1. Open your Zoho Sheet → File → Publish to Web
2. Select the sheet, choose format: CSV and click Publish
3. Copy the generated public URL
4. Paste it below and click Fetch
Use the CSV export URL from the publish dialog.
