# unnati

## Overview

**Purpose:** Job search automation and career assistant
**Stack:** Next.js 15, Prisma, Cloudflare Adapter
**URL:** https://unnati.pages.dev

- AI-powered job matching
- Resume optimization
- Application tracking
- Company research

## Tech Stack

- Next.js 15.5 with Cloudflare adapter
- Prisma ORM
- Supabase (database)
- Cloudflare Workers
