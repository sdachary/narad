# chitragupta

> Auth & Analytics Service

## Overview

**Purpose:** User authentication and usage analytics
**Stack:** Cloudflare Pages, Vite, TypeScript
**URL:** https://chitragupta.pages.dev

## APIs

- `/api/health` - Health check
- `/api/auth/*` - Authentication endpoints
- `/api/analytics/*` - Analytics endpoints

## Key Features

- JWT-based auth
- EdDSA key validation
- Usage tracking
