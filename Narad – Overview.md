# Narad – Overview

Narad is the AI chat + dashboard service. It runs on Vite and Cloudflare Pages, exposing the `/api/chat` endpoint for AI interactions.

## Key files
- `pages/_worker.js` – Cloudflare Worker entry point
- `pages/vault/` – Obsidian vault sync
- `pages/login/` – Authentication handlers
